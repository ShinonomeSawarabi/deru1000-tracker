# でる1000 tracker

**「でる1000問 演習」学習進捗トラッカー**

TOEIC対策問題集「でる1000問」の演習記録を管理するWebアプリです。
ブラウザだけで動作し、インストール不要で使えます。

🔗 **[アプリを開く](https://esawarabi.github.io/deru1000-tracker/)**

---

## 機能

- **セクション別記録** — S01〜S15（通し番号3〜659）を15セクションに分割して管理
- **周回学習対応** — 第1周・第2周…と何周でも記録を追加可能
- **各セクションの記録項目**
  - 実施日
  - 誤答数
  - 所要時間
  - 最終残誤答（復習後の残り間違い数）
- **完了チェック** — セクションごとにチェックボックスで完了管理
- **過去周回の比較** — セクション展開時に過去の周回記録をチップ表示
- **統計** — AVG ERR / AVG REM をヘッダーに常時表示
- **自動保存** — ブラウザの localStorage に自動保存（60秒ごと）

---

## 使い方

### オンラインで使う
上のリンクからアクセスするだけで使えます。

### ローカルで使う
`index.html` をダウンロードしてブラウザで開いてください。

```
deru1000-tracker/
└── index.html   ← これをブラウザで開く
```

---

## データについて

- データはブラウザの **localStorage** に保存されます
- サーバーへの送信は一切ありません
- ブラウザのキャッシュをクリアするとデータが消えるため、定期的に「💾 保存」ボタンを押してください
- 異なるデバイス間の同期には対応していません
